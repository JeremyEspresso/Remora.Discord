public override Task<Result<IMessage>> EditMessageAsync(Snowflake channelID, Snowflake messageID, Optional<string?> content = default(Optional<string?>), Optional<IReadOnlyList<IEmbed>> embeds = default(Optional<IReadOnlyList<IEmbed>>), Optional<MessageFlags?> flags = default(Optional<MessageFlags?>), Optional<IAllowedMentions?> allowedMentions = default(Optional<IAllowedMentions?>), Optional<IReadOnlyList<IAttachment>> attachments = default(Optional<IReadOnlyList<IAttachment>>), Optional<IReadOnlyList<IMessageComponent>> components = default(Optional<IReadOnlyList<IMessageComponent>>), CancellationToken ct = default(CancellationToken))

Parameters

Name Type Description
channelID Snowflake The ID of the channel the message is in.
messageID Snowflake The ID of the message.
content Optional<string?> The new content of the message.
embeds Optional<IReadOnlyList<IEmbed>> The new embeds associated with the message.
flags Optional<MessageFlags?> The new message flags.
allowedMentions Optional<IAllowedMentions?> The allowed mentions for the message.
attachments Optional<IReadOnlyList<IAttachment>> The attachments to keep, if any.
components Optional<IReadOnlyList<IMessageComponent>> The components of the message.
ct CancellationToken The cancellation token for this operation.

Type Description
